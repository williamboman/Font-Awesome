To nest multiple icons, use the fa-nest class on the parent icon. By default the nested icons will be one half the size of the parent, but this can be adjusted using the lg (for two thirds size) and sm (for one third size) classes.

Additionally, the child icon, by default, will be centered within the parent icon. Adjustments can be made using the following classes on the child icon:

  • To adjust the child icon to the left of center use align-l1, align-l2, align-l3 or align-l4
  • To adjust the child icon to the right of center use align-r1, align-r2, align-r3 or align-r4
  • To adjust the child icon above center use align-t1, align-t2, align-t3 or align-t4
  • To adjust the child icon below center use align-b1, align-b2, align-b3 or align-b4

There are several shortcuts for the above alignments.

  • Top left: align-tl is an alias for "align-t2 align-l2"
  • Top right: align-tr is an alias for "align-t2 align-r2"
  • Bottom right: align-br is an alias for "align-b2 align-r2"
  • Bottom left: align-bl is an alias for "align-b2 align-l2"
